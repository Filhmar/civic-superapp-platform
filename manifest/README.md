# Civic Platform — Logo Kit (mark 2c · "Service Grid — Bandila")

The chosen mark: the super-app service grid in Philippine-flag colors —
blue #0038A8, red #CE1126, light blue #1348C4, gold #FCD116. The rounded
gold pill with the blue dot is "the citizen moment".

## Palette
- Flag blue `#0038A8` (primary / theme color)
- Flag red `#CE1126`
- Light blue `#1348C4`
- Flag gold `#FCD116` (accent)
- Deep navy `#10214D` (dark surfaces)
- Ink `#1B2A20`, background `#F6F8F6`

## Files
- `favicon.svg` — modern scalable favicon (root)
- `svg/logo.svg` — primary mark (transparent)
- `svg/logo-reversed.svg` — for dark/navy surfaces
- `svg/logo-mono.svg` — single-ink uses (print, embossing)
- `svg/lockup.svg` — mark + CivicPlatform wordmark (Plus Jakarta Sans 800)
- `icons/` — favicon-16/32/48, icon-192/512 (+ maskable), apple-touch-icon (180), mstile-150
- `og/og-image.png` — 1200×630 social share card
- `site.webmanifest`, `browserconfig.xml`, `snippet.html` — drop-in web wiring

## Usage
1. Copy `favicon.svg`, `site.webmanifest`, `browserconfig.xml`, `icons/`, `og/` to your web root.
2. Paste the tags from `snippet.html` into `<head>`.
3. Maskable icons keep the mark inside a 62% safe zone on white — required by Android launchers.

## Rules
- Wordmark font is Plus Jakarta Sans 800 ("Civic" ink · "Platform" flag blue; reversed: white · gold).
- Clear space around the mark: ≥ 25% of its width. Minimum size 16px.
- Do not recolor tiles individually, rotate the grid, or place the colored mark on saturated backgrounds — use the reversed or mono version.
